Creates a widget of some type.

Parameters
typeThe type of window - should generally be "[YourScheme]/[Label,Button,FrameWindow,etc.]". These strings are available in the scheme files.
destRectPercentThe destination+rectangle of the widget, in terms of percentages of the screen.
destRectPixelsThe destination+rectangle of the widget, in terms of pixels.
parentThe parent window - defaults to nullptr.
nameThe name of the window - defaults to a unique identifier. Only set this if a pointer of the window cannot be retained/propagated as a handle.
Returns
A pointer to the new CEGUI::Window - can be casted to some child type generally.
