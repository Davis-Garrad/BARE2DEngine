\doxysubsection{Detailed Description}
This is the basic Lua script wrapper -\/ it is what the end-\/user will create and add to the queue. It basically just holds the text that the script will run. 

This is not to be used by the end user under any circumstances. This is the class which represents a Lua context that has been initialized with a script\textquotesingle{}s Lua to run. It holds various data, such as a handle to the parent context.
